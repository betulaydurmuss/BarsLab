# 🧪 BarsLab

BarsLab, Go programlama dili kullanılarak geliştirilmiş basit bir dinamik blog web uygulamasıdır.  
Uygulama, admin paneli üzerinden blog yazısı eklemeye ve ana sayfada bu yazıları listelemeye olanak tanır.

## 🚀 Özellikler

- Dinamik web sitesi yapısı
- Admin giriş sistemi
- Blog yazısı ekleme
- SQLite veritabanı kullanımı
- Go standart kütüphaneleri ile geliştirilmiştir

## 🔐 Admin Giriş Bilgileri

Kullanıcı adı: `admin`  
Şifre: `admin`

## 🛠️ Kullanılan Teknolojiler

- Go (Golang)
- HTML
- SQLite
- net/http paketi

## ▶️ Çalıştırma

```bash
go run main.go
http://localhost:8080
